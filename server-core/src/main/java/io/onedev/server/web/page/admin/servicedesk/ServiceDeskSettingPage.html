服务桌面功能使用户能够通过向系统发送电子邮件来创建问题。 有问题可以讨论 完全发送电子邮件，无需登录系统.

邮件设置中定义的系统电子邮件地址应为 用作此类电子邮件的收件人，项目名称可以附加到此地址，用于指示在哪里创建问题. 例如，如果系统电子邮件地址 被指定为 support@example.com, 发送电子邮件至 support+myproject@example.com 将创建 发行于 myproject. 如果未附加项目名称，系统将使用 项目指定信息如下
